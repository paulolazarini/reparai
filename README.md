# Guia Rápido: Rodando o Projeto Reparaí

### Requisito
É necessário um Mac com **Xcode** instalado.

### 1. Clonar o Projeto
Abra o aplicativo Terminal e execute os comandos:

```bash
git clone [https://github.com/paulolazarini/reparai.git](https://github.com/paulolazarini/reparai.git)
cd reparai
```

### 2. Abrir no Xcode
Na pasta reparai que você acabou de acessar, abra o arquivo Reparai.xcodeproj (ou abra direto pelo Finder).

### 3. Aguardar Dependências
Espere o Xcode baixar os pacotes do Swift (Swift Package Manager). A barra de status no topo da janela indicará o progresso.

### 4. Rodar o App
Escolha um simulador de iPhone na barra superior e clique no botão de Play ou use o atalho Cmd + R.
